Procedural Graph

Procedural graphs specify how procedural objects flow through the graph's nodes and, therefore, define the rules for procedural generation in pagoda.

Graphs are created by reading from a graph file in pagoda's own graph format.

Nodes

Name Description
Operation Defines an operation
InputInterface Routes incoming objects to an input interface of a operation node's operation.
OutputInterface Fetches the objects of an output interface of a operation node's operation.
Parameter Allows to create parameters and expressions that are set in operation nodes.
Router Filters and routes incoming procedural objects to different downstream nodes.
